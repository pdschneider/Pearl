# Supported File Types

This list contains all file types currently supported as attachments in **Pearl**.

### Document Files
- .pdf

### Text Files
- .txt
- .log
- .md
- .adoc
- .tex
- .org
- .text
- .asc
- .lst
- .list

### Data Files
- .csv
- .json
- .m3u
- .xml
- .tsv
- .pid

- .py
- .pyw
- .spec
- .sh
- .bat
- .ps1
- .htm
- .java
- .xhtml
- .html
- .css
- .rst
- .iss

- .dart
- .js
- .sql
- .go
- .rs
- .php
- .cpp
- .c
- .h
- .cs

- .ts

- .pl
- .lua
- .scala
- .hs
- .jl
- .r
- .scss
- .sass
- .less
- .vbs
- .asp
- .ipynb

### Configuration Files
- .ini
- .cfg
- .toml
- .yaml
- .conf
- .yml
