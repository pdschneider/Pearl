# Supported File Types

This list contains all file types currently supported as attachments in **Pearl**.

### Document Files
- .pdf
- .docx

### Text Files
- .txt
- .log
- .md
- .adoc
- .tex
- .org
- .text
- .asc
- .lst
- .list
- .eml

### Data Files

- .json
- .m3u
- .xml
- .tsv
- .pid

*Note: Some file types have multiple purposes. Binary, audio, 3D-graphics, and video file formats are not supported. If a file extension on this list is not working with Pearl, it may be incompatible.*

If there is a particular file type you would like to see supported, feel free to open a GitHub issue feature request!
